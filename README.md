# ROS_Installer
## 介绍
---
这个bash脚本将帮助你在Ubuntu操作系统上自动安装适合你操作系统的ROS发行版
## 如何使用
---
安装ROS
> chmod +x ROS_Installer.sh

> ./ROS_Installer.sh

卸载ROS
> chmod +x ROS_Uninstaller.sh

> ./ROS_Uninstaller.sh
## 注意
> 1.不保证100%的成功率

> 2.你需要在安装有桌面的Ubuntu操作系统上运行该脚本

> 3.为了避免一些问题，请以root身份运行该脚本

> 4.由于安装的是全功能版，请确保你有2G的可用存储空间
